# 🌙 My Neovim Configuration (LazyVim)

A personal Neovim setup built on top of [LazyVim](https://github.com/LazyVim/LazyVim). Optimized for Windows and tailored for Web and Java development.

## 🚀 Features

This configuration includes built-in support for:
- **Languages:** TypeScript/JavaScript, Java, HTML/CSS (Tailwind), and JSON.
- **Formatting:** Prettier integration for consistent code style.
- **UI:** Modern interface with `lualine`, `bufferline`, and `noice.nvim`.
- **Package Management:** `lazy.nvim` for plugins and `mason.nvim` for LSP/linters/formatters.

## 📋 Requirements

To use this configuration, ensure you have the following installed:
- **Neovim (v0.9.0+):** The editor itself.
- **Git:** For plugin management.
- **Node.js & npm:** Required for LSP servers (TS/JS, Tailwind).
- **JDK (Java Development Kit):** Required for Java support.
- **[Nerd Font](https://www.nerdfonts.com/):** Highly recommended for icons (e.g., JetBrainsMono Nerd Font).

## 📥 Installation (Windows)

1. **Backup your current config (if any):**
   ```powershell
   Rename-Item -Path "$env:LOCALAPPDATA\nvim" -NewName "nvim_backup"
   ```

2. **Clone this repository:**
   ```powershell
   git clone https://github.com/YOUR_USERNAME/nvim.git $env:LOCALAPPDATA\nvim
   ```

3. **Open Neovim:**
   ```powershell
   nvim
   ```
   Lazy.nvim will automatically download and install all plugins and language servers.

## ⌨️ Common Keymaps

LazyVim comes with excellent defaults. Here are some essentials:
- `<leader>e` - Toggle File Explorer (Neo-tree)
- `<leader>ff` - Find Files (Telescope)
- `<leader>sg` - Live Grep (Search through code)
- `<leader>l` - Open Lazy (Plugin Manager)
- `<leader>cm` - Open Mason (LSP/Linter/Formatter Manager)
- `K` - Hover Documentation
- `gd` - Go to Definition
